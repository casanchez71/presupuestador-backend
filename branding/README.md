# SOLE INTHEGROW

Primera propuesta de identidad en SVG.

Archivos:

- `sole-inthegrow-icon.svg`: isotipo para favicon, app icon o avatar.
- `sole-inthegrow-logo-horizontal.svg`: versión principal para web, encabezados y presentaciones.
- `sole-inthegrow-logo-stacked.svg`: versión vertical para portada, redes o piezas cuadradas.

Dirección visual:

- Sol: claridad, dirección y energía.
- Brote: crecimiento, ejecución y desarrollo.
- Verde oscuro `#143D34`: solidez y enfoque.
- Dorado cálido `#E0A33A`: liderazgo, valor y visibilidad.

Si quieres, la siguiente iteración puede ir en una de estas direcciones:

- más corporativo y minimalista
- más premium y editorial
- más tecnológico y moderno
- más orgánico y humano
